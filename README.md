# Cplusplus-Platforma_eStore

Toata cerinta se poate gasi pe site-ul https://ocw.cs.pub.ro/courses/poo-is/tema2 .

Obiective:
Obiectivul temei îl reprezintă exercițiul cu mai multe concepte de bază ale Programării Orientate pe Obiecte, presupunând că totul se întâmplă în contextul unui proiect pe care trebuie să îl implementați la locul de muncă si o parte din implementare a fost deja facuta de alti colegi.

Se utilizează următoarele concepte:

- Derivare și agregare
- Clase și metode virtuale
- Liste neomogene
- STL

Descriere
Prin această temă, dorim să implementam un sistem pentru o platforma online dedicata vanzarii de produse diverse. Entitatile principale sunt Produsele, Utilizatorii si Serverul care stocheaza detalii despre produse si utilizatori si unde se pot efectua diverse operatii asupra entitatilor, precum căutări de produse, de utilizatori sau modificări de diferite cantități (adăugare sau scoatere de produse din coș). Clasa Server este piesa ce leaga tot. Aici stocam toate datele legate de Useri si Produse (in liste neomogene).
